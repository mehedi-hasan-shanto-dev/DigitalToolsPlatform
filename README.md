# DigiToolsPlatform

## 📌 Project Overview

🔗 **Live Demo (Netlify):** [whimsical-entremet-c5c4f5.netlify.app](https://whimsical-entremet-c5c4f5.netlify.app)  
🔗 **GitHub Repository:** [DigitalToolsPlatform](https://github.com/mehedi-hasan-shanto-dev/DigitalToolsPlatform)

DigiTools is a modern React-based web application that enables users to explore, select, and manage premium digital resources such as AI tools, templates, and productivity software. The platform is built with a focus on clean UI, responsive design, and a seamless user experience.

---

## 🛠️ Technologies Used

- **React.js**
- **Tailwind CSS**
- **DaisyUI**
- **React Toastify**
- **Axios**
- **React Icons**

---

## 🚀 Key Features

1. **Smart Cart System**  
   Add products effortlessly with duplicate prevention and instant notifications.

2. **Dynamic Product Rendering**  
   Product data is fetched dynamically from a JSON file using Axios, ensuring scalability and flexibility.

3. **Interactive Notifications**  
   Real-time success and error alerts powered by React Toastify (e.g., item added successfully or already exists).

---

## 🎨 UI Highlights

- Gradient-driven modern design
- Fully responsive across mobile and desktop
- Smooth hover animations and interactive effects
